# sport-health

## 1. 该项目是什么？

该项目是基于Vue+Springboot前后端分离的个人运动健康项目，当前仓库为后台管理部分

## 2. 如何运行本项目？

1. 下载/clone项目代码到本地
2. 将下载的代码导入IDEA
3. 运行三个数据库文件
4. 修改application中的mysql的用户名和密码为你自己的。
5. 运行项目
